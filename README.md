# ArkRinth App
This is a fork of the Modrinth app with the following features:

- No ads at all
- No more telemetry
- Blue color scheme
- Drag-And-Drop upload for modpack resources
- Always uptodate with modrinth
- Offline account support
- Native window decoration support for fixing some bugs with the custom ones
- And lots more!

If you want to know how to build this just look at modrinths documentation.
